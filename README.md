>>> HTML:
  1.HTML, or HyperText Markup Language, is the standard language for creating and structuring web pages.
  2.It is not a programming language, but a markup language that describes the structure and semantics of content.
>>> CSS:
  1.CSS, which stands for Cascading Style Sheets, is a style sheet language used in web development to control the presentation and formatting of HTML documents.
  2.It allows developers to define the visual appearance of web pages, including aspects such as layout, colors, fonts, spacing, and the positioning of elements.
>>> JS:
  1.JavaScript is a scripting language used to create and control dynamic website content.
